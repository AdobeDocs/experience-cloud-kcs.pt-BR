## Resolução {#resolution}


Essa é uma falha no código do cliente que precisa ser resolvida atualizando a versão do nó em um módulo com falha usando o npm. Para fazer essa alteração, seu cliente precisa:

1. Encontrar a versão do nó que está sendo usada no pom.xml do Reactor. No exemplo abaixo: 16.14.0    `pom.xml`:


   ```
   configuration
           nodeVersionv16.14.0/nodeVersion. 
           npmVersion8.3.1/npmVersion
       configuration
   ```
2. Adicione esta versão do nó como <b>devDependencies</b> no arquivo `package.json` do módulo que falhou na criação. No exemplo acima, é o módulo UI Frontend.    ANTES - `package.json`:


   ```
   "devDependencies": {
       "@babel/core": "7.0.0",
       "@babel/plugin-proposal-class-properties": "7.3.3",
       "@babel/plugin-proposal-object-rest-spread": "7.3.2",
       "@material/tab-bar": "12.0.0",
       },
   ```


   DEPOIS - `package.json`:


   ```
   "devDependencies": {
       "@types/node": "16.14.0",                     <b></b>*THIS LINE WAS ADDED
       "@babel/core": "7.0.0",
   ```
