# QUE-370030 Você não tem os direitos necessários na pasta do identificador 85948572 (documentos do tipo &#39;Fluxo de trabalho (fluxo de trabalho)&#39;)

## Descrição

Foi relatado que os workflows podem falhar com o seguinte erro:<br><br> <br><br>

```
QUE-370030 You do not have the rights needed on folder of identifier 85948572 (documents of type 'Workflow (workflow)') to complete the operation.
```

<br> 

## Resolução


O motivo para isso é que o operador associado ao workflow não tem acesso de gravação a uma pasta necessária.

Para corrigir isso, verifique onde o workflow que está sendo executado está localizado (qual pasta). Para fazer isso, acesse [!UICONTROL Administração] - [!UICONTROL Configuração] - [!UICONTROL Schemas de dados] e encontre a `xtk:folder` esquema. Localize a pasta pela ID no erro que deve estar presente no [!UICONTROL Dados] do schema. Em seguida, configure a lista e exiba &quot;[!UICONTROL Nome completo]&quot; que mostrará a localização da pasta.

Vá para a pasta e clique com o botão direito do mouse nela para exibir as propriedades. Acesse a guia Security e verifique as permissões do operador.
