# [!UICONTROL File collector] alternativa de atividade no ACS

## Descrição

Como não temos a [!UICONTROL File collector] no ACS, você pode criá-lo usando as seguintes atividades e terá mais ou menos a mesma funcionalidade

## Resolução


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



O [!UICONTROL File collector] O permitirá o download de vários arquivos com base no padrão do nome de arquivo de um servidor externo. Cada arquivo é passado para o [!UICONTROL Carregar arquivo] , fornecendo dados na atualização.
Depois de concluído, ele continua com o arquivo a seguir e assim por diante.
A transferência de arquivo também tem o [!UICONTROL Configurações de histórico], como você sabe de [!UICONTROL File collector].
