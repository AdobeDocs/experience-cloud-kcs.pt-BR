# Dimension de Controle versus direcionado no Analytics


Este artigo discute a dimensão &quot;Controle versus Direcionado&quot; na integração do Adobe Analytics (A4T) para atividades do Automated Personalization/Direcionamento automático. O grupo Controle representa uma porcentagem de tráfego aleatório, enquanto o grupo Direcionado consiste no tráfego restante selecionado com base no algoritmo.

## Descrição {#description}


### Ambiente

Adobe Target

### Problema/Sintomas

Ter a integração A4T estabelecida. Você pode ver os dados de dimensões como &quot;Atividades do Target&quot;, &quot;Experiências do Target&quot; etc. no espaço de trabalho, mas você não poderá ver dados para a dimensão &quot;Controle versus Direcionado&quot;. 

Assim, você gostaria de saber sobre que tipo de dados ele captura e a razão pela qual você não está vendo nenhum dado.


## Resolução {#resolution}


A dimensão Controle versus direcionado no Analytics é para atividades de Automated Personalization/Direcionamento automático em que há uma porcentagem de tráfego aleatório - controle e o restante é tráfego direcionado com base em algoritmo
