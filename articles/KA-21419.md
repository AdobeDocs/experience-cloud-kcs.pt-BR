# A implantação do pipeline falha durante a etapa de criação

## Descrição {#description}


<b>Descrição</b>: a execução do pipeline falha durante a etapa de criação devido a erros no código ui.frontend

É possível observar mensagens de ERRO no arquivo build.log, conforme mostrado abaixo:

## Resolução {#resolution}


- localize o arquivo pom.xml na raiz do repositório Git e descubra a versão do nó,
por exemplo: nodeVersionv10.13.0/nodeVersion

- localize ui.frontend/package.json e adicione-o no objeto devDependencies já presente:
por exemplo:


```
"devDependencies": {"@types/node": "10.13.0",...}
```


Observação:
1. o “v” do nodeVersionv10.13.0 não deve ser copiado.
2. a versão do nó pode ser diferente no seu caso. Por exemplo: 12.22.7
