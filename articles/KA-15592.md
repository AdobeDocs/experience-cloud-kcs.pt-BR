# Criar uma mbox com a at.js 2.x

## Descrição


O método mboxCreate não funciona com a at.js 2.x Como podemos criar uma div de mbox no nível da página?

Eu tentei com o código abaixo, mas sem sorte :



div class=&quot;mboxDefault&quot; adobe.target.getOffer({ &quot;mbox&quot;): &quot;customTestMbox&quot;, &quot;params&quot;: { &quot;status&quot;: &quot;true&quot; }, &quot;success&quot;: function(offer) { adobe.target.applyOffer( { &quot;mbox&quot;: &quot;customTestMbox&quot;, &quot;offer&quot;: offer } ); }, &quot;erro&quot;: function(status, error) { console.log(&#39;Error&#39;, status, error); }); /div


## Resolução


Tente fechar sua tag div antes do script.

Amostra:


```
div class="mboxDefault" 
  default content to replace by offer 
/div 
script 
  var el = document.currentScript.previousElementSibling;
  adobe.target.getOffer({
    mbox: "mboxName",
    params: {
      param1: "value1",
      param2: "value2"
    },
    success: function(offer) {
      adobe.target.applyOffer({
        mbox: "mboxName",
        selector: el,
        offer: offer
      });
    },
    error: function(error) {
      console.error(error);
      el.style.visibility = "visible";
    }
  });
/script
```
