# O cliente não vê a instância de produção no New Relic

## Descrição {#description}


<b>Descrição</b> : a instância de produção do cliente não foi exibida no New Relic.


## Resolução {#resolution}


Para que o ambiente de produção apareça no New Relic, é necessário concluir um pipeline de produção.
